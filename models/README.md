# Models Directory

This directory contains various model implementations and configurations for the Artcap-Blip2 project.

## Subdirectories

- **KV/**: Key-Value modulation models
- **Gated/**: Gated cross-attention models  
- **efficientnet/**: EfficientNet-based models
- **blip2-opt-2.7b/**: BLIP2 with OPT-2.7B language model

## Usage

Each subdirectory contains its own README with specific usage instructions.
